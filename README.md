# 介绍
## 安装
1. 下载鼠须管
2. 安装
```sh
git clone https://github.com/RVK4NP3N6/Rime.git /Users/用户名/Library/Rime
```
## 特点(feature)
1. 支持拼音、五笔86、五笔拼音、数字(转人民币的填写)
2. 支持表情输入
3. 支持英文输入
4. 支持通过当前日期(date)、时间(time)、星期(week)
5. 支持自定义常用短语(custom_phrase)
6. Macos自动切换主题
7. 扩展五笔词库，常用地点及常用编程词库
